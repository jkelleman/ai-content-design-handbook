# Welcome to the Content Design Starter Template

A comprehensive resource for content designers featuring practical templates, workflows, and guidelines for creating clear, accessible, user-focused content.

## Get Started

New to content design or this template? Start here:

1. **Learn the basics** - Browse [templates](templates/) to see what's available
2. **Set up Git** - Follow the [Git Workflow guide](workflows/git-workflow.md)
3. **Write accessible content** - Review [Accessibility Guidelines](resources/accessibility-guidelines.md)
4. **Establish processes** - Implement the [Content Review Process](workflows/content-review-process.md)
