## What's in Each Section

### Templates
Practical, copy-and-customize templates for common content challenges:
- Error messages
- Success confirmations
- Empty states
- Onboarding flows
- Confirmation dialogs
- Release notes
- Microcopy (buttons, links, labels)

### Examples
Real-world content examples showing:
- Before/after improvements
- Different approaches to similar problems
- Industry best practices

### Workflows
Step-by-step guides for content processes:
- Using Git and GitHub for content
- Content review and approval
- Collaboration with engineers and designers

### Resources
Reference materials and guidelines:
- Accessibility guidelines
- Writing principles
- Content metrics
- Useful tools and extensions

### Style Guides
Templates for building your team's style guide:
- Voice and tone frameworks
- Terminology standards
- Grammar and mechanics

## How to Navigate

**If you're new to content design:**
Start with [content_design_starter.md](content_design_starter.md) and the [templates](templates/) folder

**If you're setting up workflows:**
Check out [workflows](workflows/) for Git and review processes

**If you're writing specific content:**
Browse [templates](templates/) by content type

**If you're building a style guide:**
Explore [style-guides](style-guides/) for frameworks
